# 🚗 Parking Space Availability Detection using Digital Image Processing

Parking Space Availability Detection using Image Segmentation and Spatial Filtering Techniques.
## 📌 Project Overview

This project develops an image processing pipeline to detect parking space availability from static parking lot images. By applying image segmentation, spatial filtering, thresholding, and morphological operations, the system identifies occupied and vacant parking spaces to support smart parking management.
## 🎯 Objectives

- Detect available parking spaces from parking lot images.
- Improve image quality through preprocessing techniques.
- Apply image segmentation for object extraction.
- Evaluate the effectiveness of digital image processing techniques for parking detection.
## 📊 Dataset

The project utilizes parking lot images collected for academic purposes. Images contain multiple parking slots with varying lighting conditions and vehicle occupancy.
## ⚙️ Methodology

1. Image Acquisition
2. Image Preprocessing
3. Spatial Filtering
4. Grayscale Conversion
5. Thresholding
6. Image Segmentation
7. Morphological Operations
8. Parking Space Detection
## 🖼 Image Processing Techniques

- Image Segmentation

## 📈 Results

The proposed image processing pipeline successfully distinguishes occupied and vacant parking spaces under various parking lot conditions. The combination of preprocessing, segmentation, and morphological operations improves detection accuracy while maintaining computational efficiency.
